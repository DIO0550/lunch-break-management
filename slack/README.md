## ビルド方法
1. 設定ファイルを変更する

     2. 「slackConfig.json」の値を変更する。  
     ```
     - slack-api-key => apiキーを設定する。  
     - ignore-users => 表示をしないユーザーを設定する。(Slackbotなど)
     ```
